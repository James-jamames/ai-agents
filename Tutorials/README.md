# Setup

## Windows

> python -m venv venv  
> ./venv/Scripts/activate  
> pip install -r requirements.txt  

## Ubuntu

> python3 -m venv venv  
> source ./venv/Scripts/activate  
> pip3 install -r requirements.txt
